# 🚌 Horário do Ônibus

Um aplicativo web progressivo (PWA) desenvolvido para otimizar a rotina de quem depende do transporte público para chegar ao IFPE Igarassu. Ele facilita a consulta dos horários dos ônibus de forma rápida, intuitiva e acessível, mesmo **offline**.

🔗 **Acesse o projeto online:** [horario-botafogo.vercel.app](https://horario-botafogo.vercel.app/)

---

## ✨ Funcionalidades

- **Consulta Rápida:** Visualize os horários da sua linha de ônibus com uma interface limpa e direta ao ponto.
- **Suporte Offline (PWA):** Graças à integração com Service Workers, o aplicativo continua funcionando e exibindo os horários salvos mesmo se você perder a conexão com a internet.
- **Instalável:** Pode ser instalado na tela inicial do celular (Android/iOS) ou no desktop, comportando-se como um aplicativo nativo.
- **Modo Escuro (Dark Mode):** Suporte nativo e alternância entre temas claro e escuro para maior conforto visual.
- **Design Responsivo:** Interface construída com Tailwind CSS para se adaptar perfeitamente a qualquer tamanho de tela.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído com as seguintes tecnologias:

- **[Next.js](https://nextjs.org/)** - Framework React com App Router.
- **[React](https://react.dev/)** - Biblioteca para construção da interface de usuário.
- **[Tailwind CSS](https://tailwindcss.com/)** - Estilização utilitária rápida e responsiva.
- **[next-pwa](https://github.com/DuCanhGH/next-pwa)** - Geração automática de Service Workers e configuração do Progressive Web App.
- **[Lucide React](https://lucide.dev/)** - Ícones limpos e consistentes.
- **[Vercel](https://vercel.com/)** - Hospedagem e CI/CD.
